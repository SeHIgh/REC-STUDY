기초선택자 4가지

전체선택자

*/ html 내에 등장하는 모든 형태를 선택하는 선택자(초기화값, 기본설정값 적용)(개별적 세팅 X)

태그선택자

태그이름 / html 내에 등장하는 특정한 태그를 모두 선택하는 선택자 (태그의 기본속성을 적용)

아이디선택자

본문(id="이름") / 스타일 (#이름)
특정한 이름을 부여하여 선택하는 선택자로, 하나의 이름만 부여가능 / 주로 큰단락, 제일 큰 부모박스에서 사용

클래스선택자

본문(class="이름") / 스타일 (.이름)
특정한 이름을 부여하여 선택하는 선택자로, 선택자에 여러개의 이름을 부여 가능 / 주로 작은 단락 , 변화가 잦은 요소 사용
