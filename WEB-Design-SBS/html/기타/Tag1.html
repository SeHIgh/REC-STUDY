blockquote : 긴 인용문을 나타낼 때 사용 / 들여쓰기 
pre : 태그안에서 스페이스바, 엔터, 탭키 등 이 그대로 적용되어지는 태그
/ 글자 크기와 사이즈가 기본폰트랑 다르게 표현 > 유지보수가 힘듬

hr : 수평줄을 나타내는 태그 / 단독태그 / 단락의 구분 주제가 바뀔 때/ 입체적인 형태의 선

b : 글자를 굵게 표현하고 싶을때 사용
strong : 글자를 굵게 표현하고 싶을때 사용 / 경고, 주의사항과 같이 중요한 내용을 강조할 때 사용 /리더기 사용하여 어떤 문구가 강조 되었는 지 알 수 있게함 : 웹접근성이 좋아짐

i : 이텔릭체를 표현하고 싶을 때 사용
em : 이텔릭체를 표현하고 싶을 때 사용 / 주위 텍스트에 비해 강조된 부분이나 저자의 생각을 강조 할 때 사용

q : 짧은 인용할 문구를 표현 할 때 사용 / "" 문장의 앞뒤에 표현

mark : 중요한 부분을 강조할 때 사용 / 형광펜으로 줄 그은 것 처럼 노란 배경색이 나타남

span : 스타일을 사용해서 꾸밀때 사용 / 의미가없음 ★

u : 글자 밑에 밑줄 표현 시 사용
ins : 글자 밑에 밑줄 표현 시 사용 / 문서에 나타난 텍스트를 강조 할 때 사용

s : 글자 중심에 줄 표현 시 사용
del : 글자 중심에 줄 표현 시 사용/ 문서에서 삭제된 텍스트나 지워야 할 부분 강조

small : 텍스트 작게 표현
